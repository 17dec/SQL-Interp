# SQL::Interp

Simplified interpolation of variables into SQL statements.

# Installation

To install:

```sh
perl Build.PL
./Build
./Build test
./Build install
```

Requirements:

- Sub::Exporter
- DBI (optional, needed by DBIx::Interp)

Documentation is provided in the POD and is available online at [Metacpan](https://metacpan.org/pod/SQL::Interp).
